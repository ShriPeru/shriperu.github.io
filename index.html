Selected Projects

Evaluator — Real-Time Musician Feedback System

Led a 4–5 person AI team building an on-device computer vision system that gives live feedback to cellists. Designed and trained a custom YOLO model with oriented bounding boxes for bow-string detection, then replaced the deep learning classifier with a geometric trigonometric algorithm for bow angle detection — maintaining accuracy at a fraction of the compute. Ported the pipeline to Android in Kotlin with NPU/GPU acceleration on Snapdragon chips via Qualcomm AI Hub, achieving a 3× speedup. Distributed via Firebase with AWS S3/Cognito for session data. Published in Digital Human Modeling and Applications in Health, Safety, Ergonomics and Risk Management, DHM 2026, HCII Springer Proceedings, Montreal, July 2026.

Computer Vision YOLO PyTorch TensorFlow Android / Kotlin Qualcomm AI Hub AWS
Paper (arXiv) Springer Proceedings GitHub

Weather ETL Pipeline on AWS

Built a fault-tolerant ETL pipeline on AWS using PySpark to ingest and normalize large-scale JSON weather data into PostgreSQL. Features cron scheduling on EC2, recovery logging, and SQL-based aggregation and visualization.

Python AWS PySpark PostgreSQL EC2
GitHub

Unix-like Shell & Mini C Compiler

Designed a Unix shell supporting piping, I/O redirection, subshells, wildcards, and command history with tab completion using Flex/Bison and raw terminal input. Separately built a mini compiler generating optimized x86-64 assembly from C source files with full support for arithmetic, logical, and control flow operations, handling register allocation and stack management.

C / C++ Assembly Compilers Systems Flex / Bison

GestureSynth

Built a real-time hand gesture music system using MediaPipe and OpenCV. Left hand controls guitar sounds; right hand controls piano, with gestures — swipes, pinches, sustained holds — triggering notes across both instruments. All inference runs locally via webcam with no external hardware.

Python MediaPipe OpenCV Computer Vision Real-Time
