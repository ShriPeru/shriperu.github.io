About

A background built on research, systems, and real results.

I'm Shrinand — a CS student at Purdue with minors in Math and Psychology, focused on machine learning, computer vision, and systems programming. I'm drawn to problems that sit between disciplines: where neuroscience meets signal processing, where research meets deployment, where a model has to run on a phone in real time.

My work has been recognized with two 1st place awards at Purdue's 2024 Undergraduate Research Conference. I've co-authored a paper accepted to HCI International 2026, shipped an Android app using NPU/GPU acceleration, and built everything from a Unix shell to a mini C compiler from scratch.

AI Musician Feedback System

Led a 4–5 person team building a real-time computer vision system that gives automated feedback to musicians. Trained a custom YOLO model with oriented bounding boxes for bow-string detection, then developed a trigonometric classification algorithm to detect improper bow angles — replacing a deep learning approach while maintaining accuracy.

Computer Vision YOLO PyTorch Android / Kotlin

02

Alzheimer's Diagnosis from Gene Expression

Built and evaluated ML classifiers for Alzheimer's disease diagnosis using cell-free mRNA gene expression data, achieving a Youden's index of 0.81 with 100% specificity. Led uncertainty analysis investigating feature interactions and multicollinear gene effects using KS tests and Gamma distribution simulation.

Machine Learning Scikit-learn Statistics Bioinformatics

Weather ETL Pipeline on AWS

Built a fault-tolerant ETL pipeline on AWS using PySpark to ingest and normalize large-scale JSON weather data into PostgreSQL. Features cron scheduling on EC2, recovery logging, and SQL-based aggregation and visualization.

Unix-like Shell & Mini C Compiler

Designed a Unix shell supporting piping, I/O redirection, subshells, wildcards, and command history with tab completion using Flex/Bison. Separately built a mini compiler generating optimized x86-64 assembly from C source files, with full support for arithmetic, logical, and control flow operations.
